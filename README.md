## Petshop App.

À Petshop App é uma aplicação web, para gerenciar agendamentos e serviços de um petshop,
na aplicação é possível o cliente se registrar com uma conta do google, fazer o login, escolher serviços e datas.

### Tecnologias usadas

- Next.JS 14
- Next Auth
- Next Safe Action
- React Hook Form
- Prisma
- Zod
- Tailwindcss
- Typescript
- React Icons
- PostgreSQL
